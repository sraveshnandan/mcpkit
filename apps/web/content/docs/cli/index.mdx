- `--dry-run`
- `--verbose`
- `--no-color`

And these commands:

- `init`
- `dev`
- `validate`
- `build`
- `test`
- `docs`
- `check-env`
- `doctor`
- `ship`
- `completions`

Each page in this section is based on the current command source under `packages/cli/src/commands`.
