## Purpose

`mcpkit test` shells out to Vitest.

## Basic syntax

```bash
mcpkit test
```

## Verified options

- `-w, --watch`
- `--coverage`
- `-v, --verbose`

## Examples

```bash
mcpkit test
```

```bash
mcpkit test --watch
```

```bash
mcpkit test --coverage
```

## Expected behavior

- runs `npx vitest run` by default
- runs `npx vitest` if `--watch` is set
- appends `--coverage` if requested
