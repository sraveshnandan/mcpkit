Prompts are another MCP concept that matters conceptually but is not scaffolded as a first-class template feature in the current repository.

## Current relationship to mcpkit

You can add prompt support inside the generated project code, but `mcpkit` does not currently generate dedicated prompt files or prompt-oriented helpers.

## Why the docs still include this page

A developer evaluating `mcpkit` should know where the toolkit ends today:

- it scaffolds the server project
- it helps with the workflow around the server
- it does not yet provide prompt-specific generator flows
